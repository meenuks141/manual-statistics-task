# Manual Statistics Calculation

## Objective
To manually calculate Mean, Median, Mode, Variance, Standard Deviation, and Probability using a small dataset.

## Dataset
85, 92, 76, 88, 90, 95, 68, 82, 91, 87, 78, 94

## Calculations
- Mean
- Median
- Variance
- Standard Deviation
- Probability of score greater than 90

## What I Learned
I learned how to manually calculate basic statistical measures and understood how variance and standard deviation describe data spread.
